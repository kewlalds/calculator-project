# Простий калькулятор
Навчальний проєкт для вивчення Git та Python.
## Опис
Простий консольний калькулятор, який виконує базові арифметичні операції:
- Додавання (+)
- Віднімання (-)
- Множення (*)
- Ділення (/)
## Вимоги
- Python 3.7+
## Запуск
Для запуску програми виконайте команду
python calculator.py
